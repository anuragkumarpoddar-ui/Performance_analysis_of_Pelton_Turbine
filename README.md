**🚀 Performance Analysis of Pelton Turbine using MATLAB**

📌 Project Overview

This project focuses on the performance analysis of a Pelton Turbine, an impulse turbine widely used in hydroelectric power plants. The objective was to study how different operating parameters such as load, pressure, and rotational speed affect the overall efficiency of the turbine.

The project combines experimental data analysis with data visualization using MATLAB to derive meaningful insights and performance trends.

🎯 Objectives
Analyze the effect of load variation on turbine performance
Study the relationship between rotational speed and efficiency
Evaluate the influence of inlet and outlet pressure conditions
Calculate turbine efficiency under different operating conditions
Visualize relationships using MATLAB graphs

⚙️ Methodology
Conducted experiments by varying load conditions
Collected data including:
Load (kg)
Inlet orifice pressure
Outlet orifice pressure
Inlet nozzle pressure
Turbine speed (RPM)
Efficiency (%)

Applied engineering formulas to calculate:
Jet velocity
Blade velocity
Relative velocity
Hydraulic efficiency
Used MATLAB to:
Plot graphs such as Efficiency vs Load
Analyze performance trends
Interpret system behavior under varying conditions

📊 Key Findings
As load increases, the rotational speed decreases
Efficiency increases with load up to an optimal point
Maximum efficiency achieved: ~97%
Strong correlation observed between load and turbine efficiency

📈 Visualization
Implemented MATLAB plots for:
Efficiency vs Load
Speed vs Load
Speed vs Efficiency

These visualizations helped in clearly understanding the turbine performance characteristics.

🛠️ Tools & Technologies

MATLAB (Data Visualization & Analysis)
MS Excel (Data Collection & Tabulation)
Mechanical Engineering Concepts (Fluid Mechanics & Turbomachinery)
📚 Conclusion

The project successfully demonstrates how data-driven analysis and visualization can be used to evaluate and optimize turbine performance. It highlights the importance of load management and operating conditions in achieving maximum efficiency in hydro turbines.
